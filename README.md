# 🧹 Data Cleaning & Reporting Automation

## 📌 Project Overview

This project demonstrates data cleaning, preprocessing, and automated reporting using Microsoft Power BI. The dashboard transforms raw coffee shop sales data into meaningful business insights through Power Query and interactive visualizations.

---

## 🎯 Objectives

- Clean and preprocess raw data.
- Remove duplicate records and handle data inconsistencies.
- Standardize column names and data types.
- Create automated reports using Power BI.
- Build interactive dashboards for business analysis.

---

## 📂 Dataset

**Dataset:** Coffee Shop Sales Dataset

### Dataset Includes

- Transaction ID
- Transaction Date
- Product Category
- Product Type
- Store Location
- Quantity Sold
- Unit Price
- Sales

---

### KPI Cards

### Visualizations

- 📈 Sales Trend
- 🍩 Sales by Product Category
- 📊 Top Product Types
- 🏪 Sales by Store Location
- 📦 Quantity Sold by Date

### Interactive Filters

- Product Type
- Product Category
- Month

---

## 🧹 Data Cleaning Performed

- Removed duplicate records
- Checked and handled missing values
- Corrected data types
- Standardized column names
- Created calculated Sales column

---

## 🔄 Reporting Automation

Power Query stores all transformation steps. Whenever the source dataset is updated, clicking **Refresh** automatically:

- Reapplies all cleaning steps
- Updates KPIs
- Refreshes charts and visualizations
- Generates an updated report without repeating manual work
